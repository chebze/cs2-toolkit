# SoundEspToggleService

## Purpose

Hosted service binding `Toolkit:SoundEsp:ToggleKey` (default `F5`) to toggle `SoundEspState`.

## Behavior

- Subscribes to `OnKeyPress`
- When disabled, `EnemyNoiseOverlay` skips drawing enemy and bomb ripples

## Configuration

`Toolkit:SoundEsp:ToggleKey`, `Toolkit:SoundEsp:Enabled`
