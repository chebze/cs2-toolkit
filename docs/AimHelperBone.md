# AimHelperBone

## Purpose

Target bone enum and config label parser for aim helper.

## Values

| Enum | Config | Label |
|------|--------|-------|
| `Head` | `Head` (default) | `H` |
| `Neck` | `Neck` | `N` |
| `Body` | `Body` / `Chest` | `B` |

## Key API

- `AimHelperBoneParser.Parse(string?)`
- `AimHelperBoneParser.ToLabel(AimHelperBone)`
